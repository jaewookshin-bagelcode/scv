# compact

대화가 길어지면 핵심만 남기고 군더더기를 버려 맥락을 가볍게 한다. (scv 의 자동 compaction 과
별개로, 사용자가 직접 요약을 요청할 때 쓰는 절차.)

## 보존할 것
- 내려진 **결정**과 그 이유
- 알게 된 **사실**: 파일 경로, 식별자, 명령어, 수치
- **미해결 작업**과 다음 단계
- 사용자의 **선호/제약**

## 버릴 것
- 인사·잡담·중복 설명
- 폐기된 접근·실패한 시도(결론만 한 줄로)

## 출력 형식
간결한 불릿으로 아래 섹션을 채운다. 요약 외의 말은 덧붙이지 않는다.

- 목표:
- 결정:
- 사실/경로:
- 미해결/다음 단계:
